# Command-Line Calculator 🧮

A beginner-friendly command-line calculator built in Python.

## Features
- Supports Addition, Subtraction, Multiplication, Division
- Input validation with helpful error messages
- Divide-by-zero protection
- Repeating menu for multiple calculations
- Clean code structure with separated functions

## How to Run
```bash
python command_line_calculator.py
```

## Project Structure
| Function | Purpose |
|---|---|
| add, subtract, multiply, divide | Core math logic |
| get_number, get_operator | Input handling |
| calculate | Routes to correct math function |
| show_menu, show_result | Display helpers |
| main | Controls the program loop |

## What I Learned
- Writing clean, readable Python functions
- Input validation using try/except
- Handling edge cases like divide-by-zero
- Separating logic into single-purpose functions
